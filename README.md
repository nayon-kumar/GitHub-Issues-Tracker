# GitHub Issues Tracker

<p>To show the live demo <a href="https://nayon-kumar.github.io/GitHub-Issues-Tracker/">Click Here</a></p>

## Overview

**GitHub Issues Tracker** is a web application that allows users to **view, search, and filter project issues** similar to GitHub's issue management system.  
The project fetches issue data from an API and displays it in a **clean dashboard interface** with filtering and detailed modal views.

## 🛠 Technologies Used

- **HTML5** – Application structure
- **Tailwind CSS** – Styling and layout
- **DaisyUI** – UI components
- **JavaScript (Vanilla JS)** – Logic, API calls, DOM manipulation
- **Font Awesome** – Icons

## 🚀 Features

- Login authentication system
- Display project issues dynamically from API
- Filter issues by:
  - All Issues
  - Open Issues
  - Closed Issues
- Search issues by keywords
- Issue detail modal popup
- Labels for issue types (Bug, Enhancement, Help Wanted, etc.)
- Priority indicators (High, Medium, Low)
- Responsive dashboard layout
- Loading spinner while fetching data
- "No issues found" UI state
